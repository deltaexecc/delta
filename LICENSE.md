## ❌ Troubleshooting Common Issues

**❌ Delta Executor won't open after installation**
The `.exe` may be quarantined by Windows Defender. Check `Virus & Threat Protection → Protection History` and restore the file, then add an exclusion for the Delta Executor install folder.

**Solution:** Add `C:/Program Files/DeltaExecutor/` to your antivirus exclusion list and re-download if the file was deleted.

---

**❌ "Injection failed" error on attach**
This typically happens when the Roblox client has updated but Delta Executor has not yet received its patch.

**Solution:** Close both Roblox and Delta Executor. Check for an OTA update inside the app. If none is available, wait a few hours — patches usually drop within the same day.

---

**❌ Key checkpoint page not loading**
Network issues or ad-blockers can interfere with the key verification page used by Delta Executor.

**Solution:** Disable browser ad-blockers temporarily, switch to a different network, or try completing the checkpoint on mobile data.

---

**❌ Delta Executor Apk Iphone build crashes on launch**
Sideloaded `.ipa` files need a valid signing certificate. Expired or revoked certificates will cause instant crashes on iOS.

**Solution:** Re-sign the Delta Executor Ios Ipa with a fresh certificate and re-install through your preferred sideloading tool.

---

**❌ Scripts not executing after attach**
The script may reference APIs that Roblox has deprecated, or the syntax could contain errors.

**Solution:** Test with a simple `print("Hello")` script first. If that works, the issue is with the specific script, not with Delta Executor.
